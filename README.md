Update V25. 
*เพิ่มระบบ Notification ป้ายประกาศ

คู่มือการติดตั้งระบบเช็คสต็อก SUNNY (Single File PWA)

ระบบนี้ถูกออกแบบมาให้รวมทุกอย่างไว้ในไฟล์ HTML เพียงไฟล์เดียว เพื่อความสะดวกในการดูแลรักษาและติดตั้ง

1. การเตรียมไฟล์

เปิดไฟล์ stock_checker.html ที่ได้จาก AI

คัดลอกโค้ดทั้งหมด (Select All > Copy)

สร้างไฟล์ใหม่ในคอมพิวเตอร์ของคุณ ตั้งชื่อว่า index.html

วางโค้ดที่คัดลอกลงไป แล้วกดบันทึก

2. การใช้งาน (3 รูปแบบ)

แบบ A: ใช้งานทันทีในเครื่อง (Offline Mode)

ดับเบิ้ลคลิกที่ไฟล์ index.html เพื่อเปิดผ่าน Chrome หรือ Safari ได้เลย

ข้อจำกัด: ระบบ Admin Online (ที่แก้ไขแล้วเปลี่ยนทุกเครื่อง) จะไม่ทำงาน จะใช้ได้แค่การบันทึกค่าในเครื่องตัวเองเท่านั้น

เหมาะสำหรับ: การทดสอบเบื้องต้น

แบบ B: ใช้งานออนไลน์ฟรี (แนะนำสำหรับร้านค้า)

วิธีนี้จะทำให้ระบบ Admin Online และ PWA (ติดตั้งแอป) ทำงานได้สมบูรณ์ 100%

ไปที่เว็บ Netlify Drop

ลากไฟล์ index.html ของคุณไปวางในวงกลม

ระบบจะสร้างลิงก์เว็บให้ทันที (เช่น sunny-stock.netlify.app)

ส่งลิงก์นี้ให้พนักงานใช้งานได้เลย

แบบ C: ใช้งานผ่าน Firebase Hosting (ขั้นสูง)

หากคุณใช้ Firebase อยู่แล้ว สามารถใช้คำสั่ง firebase deploy เพื่ออัปโหลดไฟล์ index.html นี้ขึ้นโฮสต์ได้เลย

3. การติดตั้งแอปในมือถือ (PWA)

เมื่อเปิดเว็บผ่านลิงก์ (แบบ B หรือ C) แล้ว:

Android: จะมีปุ่ม "ติดตั้งแอป" ในเมนู หรือแถบด้านล่างจะเด้งขึ้นมาให้กดเพิ่มลงหน้าจอหลัก

iOS: กดปุ่ม Share (แชร์) > เลือก "เพิ่มไปยังหน้าจอโฮม" (Add to Home Screen)

4. การจัดการ Admin

กดปุ่ม "Admin Mode" ที่มุมซ้ายล่างของเมนู

รหัสผ่านคือ: 123456 (สอบถามที่ fb.com/mrp.osx

หากตั้งค่า Firebase ถูกต้อง เมื่อกดบันทึก ข้อมูลจะอัปเดตไปทุกเครื่องทันที
